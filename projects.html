2 Web Application Capstone Project

Overview

This is my front-end redesign of our Capstone Project. Our project is titled "Inertia Sensor-based Back Posture Monitoring and Correction through Bespoke Exercise Suggestions". The system works by collecting data from the user's spinal curvature through a wireless device with three triaxial sensors. Once the user has data recorded into the system, the website displays exercise routines suited for the user. The suggested routine intends to rehabilitate the posture of the user.
